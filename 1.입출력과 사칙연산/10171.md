# 10171번 고양이



## 문제

아래 예제와 같이 고양이를 출력하시오.



## 입력

없음.



## 출력

고양이를 출력한다.



## 예제 출력

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```



## 제출

1.

```
a= '\\    /\\ \n )  ( \') \n(  /  ) \n \\(__)|'
print(a)
```

> 이스케이프 시퀀스 사용

* \\\ : 백슬러시(\\)

* \\' : 작은 따옴표(')
* \\" : 큰 따옴표(")
* \\n : 줄 바꿈
* \\t : 수평탭
